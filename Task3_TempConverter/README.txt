Temperature Converter Website

Description:
This project is a simple temperature converter website developed using HTML, CSS, and JavaScript.

Features:
- Converts between Celsius, Fahrenheit, and Kelvin
- Input validation
- Simple and clean UI

How to Run:
1. Open index.html in any browser
2. Enter value and select units
3. Click Convert
