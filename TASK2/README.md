Project Overview:
This project is a personal portfolio website created as part of Oasis Infobyte Level 1 Task 2.
The portfolio showcases basic information about me, my skills, and the projects I have worked on.
The goal of this task was to understand website structure, styling, and basic navigation using HTML and CSS.

Features:
- About section with basic introduction
- Skills section
- Projects section (NeighbourCart included)
- Contact section
- Simple and clean user interface

Technologies Used:
- HTML
- CSS
- (Minimal JavaScript where required)

How to Run the Project:
1. Download and unzip the project folder.
2. Open index.html in any web browser.
3. Navigate through different sections of the portfolio.
4. Open the NeighbourCart project from the Projects section.
